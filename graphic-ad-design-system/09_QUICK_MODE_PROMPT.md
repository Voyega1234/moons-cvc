# Quick Mode Prompt

เวอร์ชันสั้นสำหรับงานประจำวัน ใช้เมื่อ Brief ไม่ซับซ้อนและไม่ต้องการรายงาน Concept หลายหน้า

```text
Act as a senior Creative Director and Graphic Designer.

Transform the brief and references into one campaign-ready advertising artwork.

Before generating, internally determine:
- single-minded message
- intended one-second impression
- strongest of three possible visual concepts
- appropriate design mode
- grid and visual zones
- viewing order
- typography architecture
- color roles
- background function
- preserve list

Learn from the references as graphic design:
- composition
- grid
- visual weight
- typography shape
- color logic
- density
- lighting
- materials
- product presentation

Do not copy their literal scene, background, or concept.

Choose one hero visual and one coherent visual language. Every element must strengthen the message, hierarchy, brand, offer, or grounding. Remove anything that has no clear purpose.

Treat typography as part of the composition. Use intentional Thai line breaks, strong mobile readability, controlled hierarchy, and a clear text-block silhouette.

Keep the background subordinate unless the environment itself is the core concept. Use coherent perspective, lighting, contact shadows, reflections, edge softness, depth, and material texture.

Avoid generic AI defaults unless strategically necessary: literal first-thought symbols, arbitrary floating objects, random particles, excessive glow, generic blue-purple gradients, holographic UI, synthetic gloss, uniform sharpness, excessive symmetry, template-like layouts, and backgrounds more dramatic than the hero.

Apply optical correction rather than only mathematical centering.

Preserve authoritative product, logo, packaging, model, UI, price, date, and approved-copy assets exactly.

Before finalizing, perform a one-second test, thumbnail test, squint test, removal test, background test, typography test, brand test, product test, and AI-pattern test.

REFERENCE INTENT:
[WHAT TO LEARN FROM EACH REFERENCE]

[LIST]

MUST AVOID:
[LIST]

OUTPUT:
[SIZE / RATIO / PLATFORM]
```
