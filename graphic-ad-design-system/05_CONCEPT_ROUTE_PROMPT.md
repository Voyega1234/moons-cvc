# Concept Route Development Prompt

ใช้หลังมี Brief และ Reference Analysis แล้ว

```text
Act as a senior Creative Director developing advertising concepts from the brief and reference-design analysis below.

Do not generate the final image yet.

Do not use the first literal idea automatically. Avoid default symbols unless transformed into a brand-relevant execution.

BRIEF:
[PASTE BRIEF]

REFERENCE DESIGN DNA:
[PASTE REFERENCE ANALYSIS]

Develop three genuinely different concept routes:

ROUTE A — DIRECT COMMERCIAL
Prioritize immediate comprehension, product/offer visibility, and paid-social performance.

ROUTE B — CONCEPTUAL METAPHOR
Use one memorable visual device that communicates the message without excessive copy.

ROUTE C — EDITORIAL OR UNEXPECTED
Use a more restrained, art-directed, typographic, photographic, cultural, or unexpected approach while preserving commercial clarity.

For each route provide:

1. Route name
2. One-sentence Big Idea
3. What the audience understands in one second
4. Hero visual
5. Visual mechanism or metaphor
6. Layout archetype and grid
7. Eye path
8. Typography behavior
9. Color roles
10. Background function
11. Lighting/material approach
12. Information-density plan
13. Why it fits the brand
14. Why it does not copy the reference
15. Main risk
16. How to prevent it from looking AI-generated

Score each route from 1–5 for:

- clarity
- originality
- brand fit
- stopping power
- product/service relevance
- mobile readability
- commercial usefulness
- production feasibility
- adaptability
- low AI-cliché risk

Select one recommended route.

Then provide a concise Art Direction Blueprint for the selected route:

- canvas
- grid
- visual zones
- hero scale and placement
- headline placement and silhouette
- support information
- color system
- depth plan
- lighting
- material
- background
- preserve list
- remove list

Do not merge routes merely to include more ideas.
```
