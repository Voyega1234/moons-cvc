# Final Image Generation Prompt Template

Template นี้ใช้สำหรับเขียน Prompt เฉพาะงานหลังเลือก Concept และ Art Direction แล้ว ควรตัดหัวข้อที่ไม่เกี่ยวข้องออก

```text
INTENDED USE
Create a [FORMAT / SIZE / ASPECT RATIO] campaign-ready advertising key visual for [BRAND / PRODUCT / SERVICE].

The artwork will be used for [PLATFORM / PLACEMENT]. It must remain clear and visually distinctive at mobile-feed size.

CORE MESSAGE
[ONE SENTENCE]

INTENDED FIRST IMPRESSION
[3–5 DESCRIPTIVE WORDS]

CREATIVE CONCEPT
[DESCRIBE ONE BIG IDEA AND HOW THE VISUAL COMMUNICATES IT]

Do not introduce additional metaphors or unrelated story elements.

DESIGN MODE
Use a [EDITORIAL / PREMIUM PRODUCT / CONCEPTUAL / RETAIL PROMOTION / LIFESTYLE COMMERCIAL / TYPOGRAPHIC / COLLAGE / HYBRID] design system.

The work should feel [SPECIFIC DESIGN CHARACTER], not generically “premium.”

COMPOSITION
- underlying grid: [GRID]
- dominant axis: [AXIS]
- hero zone: [POSITION / SCALE / CANVAS SHARE]
- headline zone: [POSITION / WIDTH / ALIGNMENT]
- support/offer zone: [POSITION]
- brand/CTA zone: [POSITION]
- quiet zone: [POSITION]
- safe margin: [GUIDANCE]

Use shared alignment anchors. Apply optical correction where visual weight requires it. Avoid arbitrary placement and accidental near-tangencies.

VIEWING ORDER
1. [FIRST]
2. [SECOND]
3. [THIRD]
4. [FOURTH]

Enforce this order through scale, contrast, lighting, sharpness, color, isolation, and grouping.

HERO VISUAL
[DETAILED DESCRIPTION OF HERO SUBJECT, ACTION, SCALE, ANGLE, CROP, EXPRESSION OR FORM]

The hero must carry the highest intentional visual weight.

PRODUCT OR ASSET FIDELITY
Use the supplied [PRODUCT / LOGO / UI / MODEL] as the authoritative reference.

Preserve exactly:
- [ITEM]
- [ITEM]
- [ITEM]

Do not redraw, distort, recolor, relabel, cover, or change the proportions of authoritative assets.

SUPPORTING ELEMENTS
Include only:
- [ELEMENT AND PURPOSE]
- [ELEMENT AND PURPOSE]

Every supporting element must strengthen the message, hierarchy, brand, offer, or grounding. Do not add decorative fillers.

BACKGROUND
The background’s primary function is [CONTRAST / CONTEXT / ATMOSPHERE / METAPHOR SUPPORT / DEPTH].

[DESCRIBE ENVIRONMENT OR GRAPHIC FIELD]

Keep background [CONTRAST / SATURATION / SHARPNESS / DETAIL] below the hero. Do not make the scene more dramatic than the product or message.

TYPOGRAPHY
Use the exact approved text:

Headline:
“[EXACT TEXT]”

Supporting text:
“[EXACT TEXT]”

CTA:
“[EXACT TEXT]”

Typography direction:
- primary type category: [TYPE PERSONALITY]
- supporting type: [TYPE PERSONALITY]
- hierarchy: [SCALE RELATIONSHIP]
- line breaks: [EXACT LINE BREAKS]
- alignment: [ALIGNMENT]
- emphasis: [WORDS / COLOR / WEIGHT]
- text-block silhouette: [SHAPE]

Treat typography as part of the composition. Maintain natural Thai line breaks, readable spacing, and mobile-scale clarity. Do not fill empty areas with extra text.

If exact typography cannot be guaranteed, preserve a clean typography-safe zone and generate the base visual without invented copy.

COLOR SYSTEM
- dominant/environmental: [COLOR AND ROLE]
- structural/supporting: [COLOR AND ROLE]
- type/readability: [COLOR]
- accent/offer/CTA: [COLOR]
- neutrals: [COLOR]

Reserve the strongest contrast and saturation for [FOCAL ELEMENT]. Do not distribute maximum color intensity evenly.

LIGHTING
- key light: [DIRECTION / SIZE / SOFTNESS]
- fill: [DESCRIPTION]
- separation/rim: [IF NEEDED]
- contact shadows: [DESCRIPTION]
- reflections: [DESCRIPTION]
- color temperature: [DESCRIPTION]

Lighting must make the hero clear, grounded, and dimensional. Maintain one coherent light direction across all elements.

DEPTH AND COMPOSITING
- foreground: [DESCRIPTION]
- hero plane: [DESCRIPTION]
- supporting plane: [DESCRIPTION]
- background: [DESCRIPTION]
- focus hierarchy: [DESCRIPTION]
- overlap/occlusion: [DESCRIPTION]

All assets must share consistent perspective, edge softness, atmospheric depth, lighting, and shadow behavior.

MATERIAL AND REALISM
Describe category-appropriate surfaces:
- [MATERIAL + MICROTEXTURE]
- [MATERIAL + REFLECTION BEHAVIOR]
- [MATERIAL + IMPERFECTION]

Use controlled natural variation. Avoid oversmoothed skin, textureless surfaces, excessive gloss, synthetic perfection, and uniform sharpness.

DENSITY
The visually active area is [AREA].
The information area is [AREA].
The atmospheric area is [AREA].
The intentionally quiet area is [AREA].

Do not spread details evenly across the canvas.

CONSTRAINTS
Do not include:
- [RELEVANT EXCLUSION]
- [RELEVANT EXCLUSION]
- [RELEVANT EXCLUSION]

Avoid generic AI defaults unless conceptually necessary: arbitrary floating objects, unrelated particles, excessive bloom, generic blue-purple gradients, meaningless holographic UI, excessive symmetry, random icon clutter, and dramatic backgrounds that compete with the hero.

FINAL QUALITY
Create one cohesive, finished commercial artwork in which every visible decision feels selected, constrained, composed, and refined for this campaign.

The result must not feel like a raw AI render, stock template, decorative mood image, or a finished scene with text added afterward.
```
