# แผนสร้างงานโฆษณาคุณภาพระดับ Production

## Phase 0 — Asset Preparation

เตรียมสิ่งต่อไปนี้ก่อนเริ่มงาน:

- Brand guideline หรือ CI ที่ใช้งานจริง
- Logo แบบโปร่งใส
- Product packshot ความละเอียดสูง
- Exact copy ที่ผ่านการอนุมัติ
- ราคา โปรโมชั่น วันที่ และเงื่อนไข
- Reference ที่แสดงรสนิยมและคุณภาพเป้าหมาย
- ตัวอย่างงานเดิมของแบรนด์
- ข้อห้ามด้านแบรนด์และโฆษณา

แยกข้อมูลเป็น 3 กลุ่ม:

1. **Must preserve** — ห้ามเปลี่ยน เช่น Logo, Packaging, สีผลิตภัณฑ์, ข้อความราคา
2. **Can interpret** — ตีความได้ เช่น ฉาก, Metaphor, มุมกล้อง, Layout
3. **Must avoid** — สิ่งที่ไม่ต้องการ เช่น คนเป็น Hero, ฉาก Cyberpunk, Graphic แน่นเกินไป

## Phase 1 — Brief Diagnosis

ก่อนคิดภาพ ให้ตอบคำถาม:

- งานนี้ต้องการให้คนเข้าใจอะไรภายใน 1 วินาที?
- การกระทำที่ต้องการหลังเห็นภาพคืออะไร?
- อะไรคือ Proof, Offer หรือ Benefit ที่สำคัญที่สุด?
- งานนี้ต้องสร้างความรู้สึกแบบใด?
- อะไรคือความเสี่ยงต่อการสื่อสารผิด?
- ถ้าต้องเหลือเพียง Hero และ Headline งานยังสื่อสารได้หรือไม่?

ผลลัพธ์ที่ต้องได้:

- Single-minded message 1 ประโยค
- Desired first impression 3–5 คำ
- Priority order ของข้อมูล
- Essential elements
- Removable elements

## Phase 2 — Reference Forensics

อย่าอธิบายแค่ว่า Reference “ดูพรีเมียม” หรือ “ใช้สีสด” ให้ถอดโครงสร้าง:

- Design category
- Layout archetype
- Grid และ Alignment anchors
- Visual center of gravity
- Eye path
- Active/quiet zones
- Typography architecture
- Color roles
- Hero scale
- Depth plan
- Lighting logic
- Material language
- Background function
- Reusable DNA
- Content-specific elements ที่ห้ามคัดลอก

ผลลัพธ์คือ Design DNA ที่นำไปประยุกต์กับ Brief ใหม่ได้

## Phase 3 — Concept Development

สร้างอย่างน้อย 3 Route ที่แตกต่างกันจริง:

1. **Direct Commercial** — เข้าใจง่าย เหมาะกับ Performance
2. **Conceptual Metaphor** — มี Visual Idea ที่จดจำได้
3. **Editorial/Unexpected** — ภาษาภาพเรียบหรือแปลกใหม่ขึ้น

ห้ามเปลี่ยนเพียงสีหรือมุมกล้องแล้วเรียกว่าเป็นคนละ Concept

ให้คะแนนแต่ละ Route 1–5 ในหัวข้อ:

- Message clarity
- Originality
- Brand fit
- Product prominence
- Visual stopping power
- Mobile readability
- Production feasibility
- Adaptability to other ratios
- Risk of looking AI-generated

เลือก Route ที่มีเหตุผลแข็งแรงที่สุด ไม่ใช่ Route ที่มีรายละเอียดเยอะที่สุด

## Phase 4 — Art Direction Blueprint

กำหนดก่อน Generate:

### Composition

- Canvas ratio และ safe margins
- Grid type
- Hero position และ scale
- Text zone
- Product/offer/CTA zones
- Foreground, middle ground, background
- Eye path 1 → 2 → 3 → 4

### Typography

- Font category และ personality
- Display/body relationship
- Line-break plan
- Text block silhouette
- Alignment
- Weight, width, leading และ tracking
- Highlight words

### Color

- Dominant environmental color
- Supporting structural color
- Readability color
- Accent/CTA color
- Neutral colors
- Saturation hierarchy

### Lighting and Material

- Key light direction
- Fill and rim light
- Contact shadow
- Reflection behavior
- Surface texture
- Category-appropriate imperfections

### Density

- Active zone
- Information zone
- Atmospheric zone
- Quiet zone

## Phase 5 — Base Visual Generation

ถ้างานซับซ้อน ให้ Generate เฉพาะ Base Visual ก่อน:

- Environment
- Hero object/model
- Product stage
- Lighting
- Shadows
- Depth
- Negative space

ยังไม่ต้องใส่ Copy, Logo, Price และ Packshot ที่ต้องตรง 100%

## Phase 6 — Commercial Assembly

นำส่วนสำคัญมาประกอบ:

- Exact product packshot
- Approved logo
- Headline
- Offer and price
- Benefits
- CTA
- Legal/disclaimer
- Platform logo หรือ certification เมื่อจำเป็น

ตรวจ Alignment, Crop, Edge, Contact Shadow, Perspective และ Color Match ให้ทุก Asset ดูอยู่ในโลกเดียวกัน

## Phase 7 — Art Director QA

ตรวจอย่างน้อย:

- One-second test
- Thumbnail test
- Squint/blur test
- Grayscale test
- Removal test
- Background dominance test
- Brand-without-logo test
- Product fidelity test
- Typography silhouette test
- AI cliché test
- Crop adaptability test

## Phase 8 — Surgical Revision

แก้ทีละกลุ่มและย้ำ Preserve List ทุกครั้ง เช่น:

- ลด Contrast ของ Background เท่านั้น
- ปรับ Headline ลง 10% เท่านั้น
- เพิ่ม Contact Shadow ใต้สินค้าเท่านั้น
- เปลี่ยน Font Treatment แต่รักษา Layout และ Hero

หลีกเลี่ยงคำสั่งกว้าง เช่น “ทำให้สวยขึ้น” เพราะเปิดโอกาสให้โมเดลแก้ทั้งภาพ

## Phase 9 — Production Export

ตรวจ:

- ขนาดไฟล์และ Aspect ratio
- Safe area ของแต่ละ Placement
- ความถูกต้องของข้อความ ราคา และวันที่
- Logo clear space
- Product label
- Mobile readability
- สีและ Contrast
- Version naming

แนะนำให้เก็บอย่างน้อย:

- Master artwork
- Textless base visual
- Final copy version
- Alternate crop
- Revision prompt ล่าสุด
